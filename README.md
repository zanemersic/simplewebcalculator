# Simple Web Calculator

A straightforward, interactive web calculator built using vanilla HTML, CSS, and JavaScript. It features a modern dark UI with a vibrant radial gradient background and performs standard mathematical operations.

## Features

- **Standard Arithmetic:** Supports addition, subtraction, multiplication, and division.
- **Continuous Calculation:** String multiple operations together without needing to press equals every time.
- **Clear Display:** Dedicated 'C' button to reset the current calculation.
- **Responsive UI:** Button hover and active states for tactile user feedback.
- **Grid Layout:** Structured using modern CSS Flexbox and Grid for precise alignment.

## File Structure

- `index.html`: The main structural file containing the calculator layout.
- `style.css`: Contains all styling, including the dark theme and grid system.
- `script.js`: Handles the underlying calculator logic and DOM updates.
- `images/favicon.png`: The website icon.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

Since this is a vanilla web project, no build tools or package managers are required.

1. Clone the repository:
   ```bash
   git clone [https://github.com/zanemersic/simplewebcalculator.git](https://github.com/zanemersic/simplewebcalculator.git)
   ```
2. Navigate to the project directory.
3. Open the `index.html` file directly in your web browser.
